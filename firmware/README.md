# Firmware

This directory contains the software running on the Raspberry Pi.

## Requirements
- Raspberry Pi OS (Lite or Desktop)
- Python 3.x
- Required libraries: TBD

## Installation
```bash
git clone https://github.com/StePol/MultiChannel_datalogger.git
cd MultiChannel_datalogger/firmware
pip install -r requirements.txt
```

## Usage
```bash
python main.py
```
